# Modern Bakery

Responsive landing page for a bakery, built from Figma design.

Built with focus on semantic HTML, clean SCSS structure, and responsive design.

## Features
- Responsive layout (mobile, tablet, desktop)
- SCSS
- Vanilla JavaScript
- Modal window
- Burger menu
- Smooth animations

## Technologies
- HTML5
- SCSS
- JavaScript
